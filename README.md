# SimplePageIndicator

Flexible dot indicator to use in Android without Viewpager.

####  Add the JitPack repository to your build file
```
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```
#### Add dependency
```
dependencies {
    implementation 'com.github.renanalmeida:SimplePageIndicator:Tag'
}
```
